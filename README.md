# Dotfiles

#### Clone

```bash
mkdir -p ~/Code/oss
git clone git@github.com:micburks/dotfiles.git ~/Code/oss/dotfiles
cd ~/Code/oss/dotfiles
```

#### Install

```bash
./install.sh
```
